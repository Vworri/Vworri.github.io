Url Shortener Microservice

Shorten Long Urls

This microservice takes urls with the forms http://something.com, https://something.com or something.com. The microservice tests the urls and if the response is not undefined or the status code is not 404, the service will shorted the link and return it to the same page. You can check out the api by sending a post request to the route /api. The microservice will not generate a new url if a record of the original url exists in the database. This utilized mongodb and is deployed on Heroku. If you want to try it out, I welcome you to View Project .

What I used:React, Heroku, Marked, MongoDb
